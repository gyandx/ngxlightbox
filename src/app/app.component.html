Ngx Light Box

Ngx light box is used to preview images in popup zoom manner

Steps to use ngx-light-box in angular

  1. npm install --save ngx-lightbox

    2. update styles in angular.json and insert this in styles array

  2. "./node_modules/ngx-lightbox/lightbox.css"

    3. in app.module import lightmodule in the import array

  3. imports: [ LightboxModule ]

    4. For further reference click here:- NgxLightBox
